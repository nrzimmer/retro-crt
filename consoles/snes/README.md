# Super Nintendo / Super Famicom — Mods

---

## SNES Mini (Super Famicom Jr.) — csync

SNES Mini não expõe csync no MultiAV. Para usar cabo SCART com csync limpo:

- csync disponível no **pino 18 do chip S-RGB** (PPU2 / S-RGB encoder) na placa
- Requer acesso interno e fio soldado até o conector MultiAV ou diretamente ao cabo

Alternativa sem mod: usar luma (pin 7 do MultiAV) como sync — funciona com a maioria dos displays.
