# Consoles — Mods Internos

Mods de hardware nos consoles para habilitar ou melhorar saída RGB/áudio.  
Cabos SCART correspondentes estão em [`scart/cables/`](../scart/cables/).

| Pasta | Console | Mods documentados |
|-------|---------|-------------------|
| [`nes/`](nes/) | NES / Famicom | NESRGB, PPU-LITE (open source), áudio stereo, expansion audio |
| [`n64/`](n64/) | Nintendo 64 | RGB amp (THS7316), opções de mod analógico |
| [`snes/`](snes/) | Super Nintendo | SNES Mini — csync no chip S-RGB |
| [`megadrive/`](megadrive/) | Sega Mega Drive | Mod stereo áudio (MD1) |
