# Cabo SCART RGB — Nintendo 64
> ⚠️ N64 NTSC não tem saída RGB nativa. Mod obrigatório antes de usar este cabo.  
> Ver [`consoles/n64/README.md`](../../consoles/n64/README.md) para opções de mod.

---

## Conector MultiAV — mesmo do SNES

**Idêntico ao SNES** — mesmo conector card-edge 12 pinos, mesma pinagem.

| Pin | Sinal |
|-----|-------|
| 1 | Red (após mod) |
| 2 | Green (após mod) |
| 3 | CSync (NTSC) / +12V (PAL) |
| 4 | Blue (após mod) |
| 5 | GND |
| 6 | GND |
| 7 | Y / Luma |
| 8 | C / Chroma |
| 9 | CVBS |
| 10 | +5V |
| 11 | Audio L |
| 12 | Audio R |

---

## Cabo após mod RGB — igual ao SNES NTSC

Após instalar mod RGB, o cabo é **idêntico ao cabo SNES NTSC**:

```
MultiAV                  Componentes              SCART
────────                 ───────────              ─────
Pin 1  (R)  ──── C+ 220µF C- ───────────────────► Pin 15 (Red In)
Pin 2  (G)  ──── C+ 220µF C- ───────────────────► Pin 11 (Green In)
Pin 4  (B)  ──── C+ 220µF C- ───────────────────► Pin 7  (Blue In)
Pin 3  (CSync) ──────────────────────────────────► Pin 20 (Sync In)
Pin 10 (+5V) ── 820Ω ─┬─ 1kΩ ─ GND ─────────────► Pin 16 (Blanking)
Pin 11 (Audio L) ───────────────────────────────► Pin 6  (Audio In L)
Pin 12 (Audio R) ───────────────────────────────► Pin 2  (Audio In R)
Pin 5/6 (GND) ──────────────────────────────────► Pins GND
```

Caps 220µF nas linhas RGB (mesmo DC offset do SNES).

## Diferenças em relação ao SNES

| Aspecto | SNES | N64 |
|---------|------|-----|
| RGB nativo (NTSC) | ✓ Sim | ✗ Requer mod |
| Conector MultiAV | Idêntico | Idêntico |
| Cabo RGB SCART | ✓ direto | Só após mod |
| Caps 220µF no cabo | ✓ Sim | ✓ Sim (mesmo) |
| csync no pin 3 | ✓ (exceto 1CHIP-03) | Depende da revisão |
| S-Video nativo | ✓ | ✓ NTSC sim |
| Blur interno | — | N64 tem blur nativo — mod adicional para remover |
