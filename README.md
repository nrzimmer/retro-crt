# CRT — Setup de Videogames em CRT

Documentação para uso e modificação de TV CRT Sony Trinitron KV-21FS140 com consoles retro via RGB SCART.

## Estrutura

```
tv/          — tudo sobre a KV-21FS140 (manuais, circuitos, mod, reparo)
scart/       — padrão SCART, switches, cabos por console
consoles/    — mods internos dos consoles (RGB, áudio, etc.)
```

| Pasta | Conteúdo |
|-------|----------|
| [`tv/`](tv/) | Manual de serviço, análise de circuitos, mod RGB, reparo pincushion |
| [`scart/`](scart/README.md) | Padrão SCART, switch, cabos por console, fornecedores |
| [`consoles/`](consoles/) | Mods de consoles (NES NESRGB/PPU-LITE, N64 RGB, SNES Mini, MD stereo) |
